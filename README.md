# Security Guard Bot

بوت حماية ديسكورد شامل مع لوقات عربية كاملة.

## الميزات

- حماية تلقائية من السبام (حذف الرسائل السريعة المتكررة).
- حماية من الروابط والإعلانات للأعضاء غير الإداريين.
- كشف Raid عند دخول عدد كبير خلال وقت قصير.
- أوامر إدارة:
  - `!setup-logs` لإنشاء كل قنوات اللوق المطلوبة.
  - `!lockall` لقفل كل الرومات الكتابية.
  - `!unlockall` لفتح كل الرومات الكتابية.
  - `!help` لعرض الأوامر.

## المتطلبات

- Node.js 20 أو أحدث.
- بوت Discord مع Token صحيح.

## التشغيل المحلي

1. تثبيت الحزم:

   ```bash
   npm install
   ```

2. نسخ ملف البيئة:

   ```bash
   copy .env.example .env
   ```

3. تعديل `.env` ووضع التوكن:

   - `BOT_TOKEN`
   - `PREFIX` (اختياري)

4. تشغيل البوت:

   ```bash
   npm start
   ```

## الصلاحيات المطلوبة للبوت

في Discord Developer Portal فعّل:

- `MESSAGE CONTENT INTENT`
- `SERVER MEMBERS INTENT`
- `PRESENCE INTENT` (اختياري)

وأعطه صلاحيات مناسبة داخل السيرفر:

- Administrator (الأسهل)
أو على الأقل:
- Manage Channels
- Manage Roles
- Moderate Members
- View Audit Log
- Read/Send Messages

## النشر على Railway (تشغيل دائم 24/7)

1. ادخل [Railway](https://railway.app/) وسجّل دخول.
2. اختر `New Project`.
3. ارفع هذا المشروع (Deploy from GitHub يفضل).
4. من تبويب `Variables` أضف:
   - `BOT_TOKEN=...`
   - `PREFIX=!` (اختياري)
5. تأكد أمر التشغيل:
   - Start Command: `npm start`
6. Railway سيشغّل البوت بشكل دائم حتى لو جهازك مطفي.

> ملاحظة: إذا البوت وقف بسبب خطأ، Railway يعيد تشغيله تلقائيا بعد إصلاح الخطأ/تحديث الكود.
